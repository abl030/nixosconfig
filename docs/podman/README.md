# Podman Documentation

This directory is the canonical location for Podman stack behavior, evidence, and historical records.

## Current Entry Point

- Current operating state: `docs/podman/current/state.md`

## Layout

- Decisions: `docs/podman/decisions/`
- Research: `docs/podman/research/`
- Incidents and empirical tests: `docs/podman/incidents/`
- Historical plans/audits/prompts: `docs/podman/archive/`

## Documentation Rule

- `current/` documents what is presently true.
- `decisions/`, `research/`, and `incidents/` record rationale and evidence.
- `archive/` is historical context and may contain superseded material.
